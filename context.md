# 프로젝트 컨텍스트: 오토파일럿 지능형 음성 배속 보정 이식 완료

## 1. 개요
오토파일럿 시스템의 안정적인 가동과 제작 비용 절감을 위해 설계했던 '지능형 음성 배속 보정' 로직을 `video-metadata` 파이프라인에 성공적으로 이식했습니다.

## 2. 작업 완료 사항

### 2.1 지능형 음성 배속 보정 (Voice Duration Scaling) 구현
- **위치**: `app/api/autopilot/video-metadata/route.ts` 및 `lib/api/server/voiceServerAPI.ts`
- **구현 로직 (Ratio-Based Scaling)**:
    - **Case A ($T_{orig} \le 32s$)**: 보정 없음 (품질 최우선)
    - **Case B ($32s < T_{orig} \le 34.5s$)**: **30.0초 정밀 타격** ($R = 30.0 / T_{orig}$) -> 비용 최적화 ($1.80)
    - **Case C ($T_{orig} > 34.5s$)**: **최대 1.15배속 제한** ($R \approx 0.8696$) -> '랩 현상' 방지 및 품질 마지노선 사수
- **기술적 특징**:
    - Replicate `ffmpeg-sandbox-2` 모델의 `atempo` 필터를 활용한 고품질 음성 변조.
    - 음성 배율($R$)을 자막 타임라인(`startSec`, `endSec`) 전체에 일괄 적용하여 싱크 오차 0% 달성.
    - `seriesId` 기반의 임시 파일 관리 시스템 구축으로 안정적인 프로세스 확보.

### 2.2 실시간 검증 (Live Log Check)
- **입력**: 32.43초 음성 생성 확인.
- **판정**: Case B (30초 타겟 구간) 진입.
- **결과**: **Ratio 0.9252 적용 -> 30.00초 최종 음성 확보** (성공).
