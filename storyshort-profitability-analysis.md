# StoryShort AI 수익성 분석 보고서

## 1. 서비스 개요

StoryShort AI는 AI를 활용하여 자동으로 쇼트폼 비디오를 생성하는 SaaS 서비스입니다. TikTok과 YouTube Shorts에 자동 업로드 기능을 제공하며, 월 구독 모델로 운영됩니다.

### 1.1 가격 정책

|------|---------|------------|--------|--------|-------------|
| **Starter** | $39 | 40개 | 400 | 1개 | $0.975 |
| **Growth** | $69 | 120개 | 1,200 | 2개 | $0.575 |
| **Influencer** | $129 | 240개 | 2,400 | 3개 | $0.537 |
| **Ultra** | $199 | 500개 | 5,000 | 4개 | $0.398 |

### 1.2 제공 기능
- AI 생성 콘텐츠
- 음성 해설 (Voiceovers)
- 배경 음악
- 워터마크 제거
- TikTok/YouTube 자동 게시

## 2. 원가 구조 분석

### 2.1 작업 단계별 사용 모델 및 비용

#### 단계 1: 스크립트 생성
**사용 모델: GPT-5 nano**
- **모델 특징**: OpenAI의 가장 빠르고 비용 효율적인 GPT-5 버전
- **용도**: 30초 쇼트폼 영상 스크립트, 제목, 설명 생성
- **토큰 사용량**: 
  - 입력: ~500 토큰 (프롬프트)
  - 출력: ~2,000 토큰 (스크립트 + 메타데이터)
  - 총: ~2,500 토큰
- **예상 비용**: $0.001-0.002 per video

#### 단계 2: 음성 합성 (TTS)
**사용 모델: TTS-1**
- **모델 특징**: OpenAI의 속도 최적화 TTS 모델
- **용도**: 스크립트를 자연스러운 음성으로 변환
- **처리량**: 30초 음성 = 약 300-400 characters
- **가격**: $15 per 1M characters
- **비용**: $0.005 per video

#### 단계 3: 비주얼 생성
**사용 모델: DALL-E 3 (선택적) + 스톡 이미지**
- **모델 특징**: OpenAI의 이전 세대 이미지 생성 모델 (안정적이고 비용 효율적)
- **용도**: 영상에 필요한 이미지 생성
- **전략**: 
  - 50% 스톡 이미지 활용: $0.01
  - 50% DALL-E 3 생성: $0.02-0.04
- **평균 비용**: $0.015 per video

#### 단계 4: 영상 편집 및 렌더링
**사용 도구: 자체 개발 자동화 시스템**
- **처리 내용**: 
  - 템플릿 기반 영상 조합
  - 자막 추가
  - 트랜지션 효과
  - 배경음악 믹싱
- **비용**: $0.008 per video (서버 컴퓨팅 비용)

#### 단계 5: 저장 및 배포
**인프라 비용**
- **CDN**: $0.002 per video
- **스토리지**: $0.001 per video
- **API 오버헤드**: $0.002 per video
- **총 비용**: $0.005 per video
